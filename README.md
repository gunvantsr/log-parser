## Local Setup

1. Clone Repository

```
git clone https://github.com/gunvantsr/log-parser.git
```

2. Install Dependancies

```
yarn install
```

3. Serve UI

```
yarn nx run app-ui:serve --configuration=development
```

4. Serve APIs

```
yarn nx run app-api:serve --configuration=development
```
